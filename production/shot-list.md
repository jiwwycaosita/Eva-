# EVA - Liste de Plans Détaillée (Shot List)
## SCÈNE 1 : Montréal, Avant la Chute

**Durée totale de la scène** : 10-15 minutes  
**Nombre total de plans** : 45-50 plans

---

## SÉQUENCE 1.1 : Vue aérienne et introduction d'Anabelle

### PLAN 1 : Vue aérienne de Montréal
- **Type** : Plan d'établissement (Establishing Shot)
- **Cadrage** : Très grand ensemble (Extreme Wide Shot)
- **Mouvement** : Travelling aérien descendant (drone)
- **Durée** : 15 secondes
- **Description** : Vue aérienne de Montréal en hiver, fin d'après-midi. Rues enneigées, immeubles résidentiels. La caméra descend lentement vers un immeuble jaune modeste.
- **Lumière** : Naturelle, crépusculaire, tons bleus/gris
- **Son** : Vent froid, bruits de ville lointains
- **Voix-off** : "Avant, je voyageais. Je parlais à des gens. Je vivais."
- **Équipement** : Drone DJI (ou location)
- **Notes** : Plan crucial pour établir l'isolement dans la ville

---

### PLAN 2 : Fenêtre de l'appartement (extérieur)
- **Type** : Plan d'ensemble
- **Cadrage** : Ensemble
- **Mouvement** : Fixe
- **Durée** : 3 secondes
- **Description** : Fenêtre d'un appartement au 3e étage, lumière chaude à l'intérieur, contraste avec le froid extérieur.
- **Transition** : Coupe franche vers intérieur

---

### PLAN 3 : Anabelle à la table de cuisine (plan large)
- **Type** : Plan d'ensemble intérieur
- **Cadrage** : Plan américain (de la taille aux pieds)
- **Mouvement** : Fixe ou léger travelling avant
- **Durée** : 5 secondes
- **Description** : Anabelle assise à une table de cuisine encombrée, écrit frénétiquement. Lumière naturelle de la fenêtre à droite, ordinateur portable ouvert.
- **Lumière** : Naturelle + lumière d'écran d'ordinateur
- **Son** : Stylo qui gratte, clavier
- **Notes** : Établir son espace de travail chaotique

---

### PLAN 4 : Gros plan - Détails sur la table
- **Type** : Insert / Détail
- **Cadrage** : Gros plan (Close-up)
- **Mouvement** : Lent travelling latéral
- **Durée** : 4 secondes
- **Description** : Travelling sur les objets de la table : verre de vin à moitié vide, tasse de café froid, flacon de pilules (antidépresseurs), papiers annotés.
- **Notes** : Suggérer sa détresse psychologique sans dialogue

---

### PLAN 5 : Gros plan - Visage d'Anabelle
- **Type** : Gros plan
- **Cadrage** : Serré sur le visage
- **Mouvement** : Fixe
- **Durée** : 3 secondes
- **Description** : Visage d'Anabelle concentré, cernes sous les yeux, cheveux négligés. Elle se frotte les tempes.
- **Lumière** : Latérale douce (fenêtre)
- **Émotion** : Fatigue, résignation
- **Notes** : Premier vrai regard sur son état mental

---

### PLAN 6 : Plan sur l'écran d'ordinateur
- **Type** : Insert / POV
- **Cadrage** : Plan serré
- **Mouvement** : Fixe
- **Durée** : 5 secondes
- **Description** : Écran d'ordinateur montrant un document Word. Titre visible : "Disparitions d'enfants : Les parents, principaux responsables ?" Elle tape, puis s'arrête.
- **Son** : Clavier qui s'arrête brutalement
- **Notes** : **IRONIE TRAGIQUE** - Préfigure la disparition d'Eva

---

### PLAN 7 : Flashback - Anabelle jeune et heureuse (insert rapide)
- **Type** : Flashback / Insert
- **Cadrage** : Plan moyen
- **Mouvement** : Léger flou, effet de mémoire
- **Durée** : 2 secondes
- **Description** : Image désaturée mais plus chaude : Anabelle (28 ans) souriante dans un avion, puis dans une salle d'interview avec micro.
- **Effet** : Surexposition légère, transition douce
- **Son** : Musique nostalgique (note de piano)
- **Notes** : Contraste avec sa vie actuelle

### PLAN 8 : Retour au présent - Anabelle boit une gorgée de vin

- **Cadrage** : Plan poitrine
- **Mouvement** : Fixe

## SÉQUENCE 1.2 : Eva joue dehors (vue de la fenêtre)

## NOTES DE RÉALISATION

### Palette de couleurs
- **Intérieur** : Tons chauds mais désaturés (beige, brun clair)
- **Extérieur** : Bleus froids, blancs neige

### Tempo
- Rythme délibéré, pas pressé
- Laisser respirer les silences
- Montage final du déménagement plus rapide (contraste)

### Direction d'actrice (Anabelle)
- Fatigue physique et mentale visible
- Micro-expressions de culpabilité
- Regard souvent fuyant ou absent
- Gestes mécaniques (automatismes de la dépression)

### Direction d'actrice (Eva)
- Enfant trop calme, résignée
- Joue seule sans plaintes
- Accepte les décisions sans protester
