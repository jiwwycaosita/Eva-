# Gabarits de prompts — Projet EVA

Ce document liste les gabarits de prompts utilisés pour construire les scènes du projet EVA. Chaque gabarit définit une structure narrative, des contraintes spécifiques et un exemple d'utilisation.

---

## 2A — Escalier de tension

### Structure
Scène en **5 paliers progressifs** :
1. **Normalité froide** : situation banale, réaliste
2. **Dissonance** : élément sonore ou visuel légèrement décalé
3. **Répétition** : le même élément revient, amplifié
4. **Faux soulagement** : explication rationnelle apparente
5. **Crochet final** : détail inexpliqué qui relance la tension

### Contraintes
- Aucun surnaturel explicite
- Progression graduelle obligatoire
- Le crochet final reste ouvert (pas de résolution)

### Exemple d'utilisation
**Prompt :**
> Écrire une scène extérieure (3 min) où Anabelle cherche son chat dans le jardin enneigé. Utiliser le gabarit 2A (Escalier de tension). Palier 1 : Appel du chat, silence. Palier 2 : Miaou lointain. Palier 3 : Miaou se rapproche puis s'éloigne. Palier 4 : Chat retrouvé sous la haie. Palier 5 : Bruit sourd provenant du ponceau voisin.

## 2B — Retardement de révélation

### Structure
Information presque révélée **3 fois** puis retirée à chaque fois :
1. **Approche 1** : personnage s'approche de la révélation → interruption réaliste
2. **Approche 2** : nouvelle tentative → nouvelle interruption
3. **Approche 3** : quasi-révélation → coupure avant confirmation

### Contraintes
- Chaque interruption doit être causalement justifiée
- Tension croissante à chaque tentative
- Aucune révélation finale dans la scène

### Exemple d'utilisation
**Prompt :**
> Scène où Anabelle tente de sortir d'un tunnel partiellement effondré (gabarit 2B). Tentative 1 : elle pousse la neige → avalanche bloque. Tentative 2 : elle trouve un passage → bruit de camion, recul. Tentative 3 : lumière visible → noir complet.

## 3A — Peur domestique

### Structure
Scène de peur dans un **lieu ordinaire** transformé en source d'angoisse :
- **Élément déclencheur** : bruit unique, objet déplacé, lumière changeante
- **Répétition** : l'élément revient de manière irrégulière
- **Amplification sensorielle** : sons, odeurs, textures deviennent oppressants
- **Absence de menace visible** : tout reste hors-champ ou suggéré

### Contraintes
- Lieu banal (cuisine, grenier, tunnel, route)
- Processus physiques réalistes uniquement
- Pas de présence humaine ou créature

### Exemple d'utilisation
**Prompt :**
> Scène dans le grenier (gabarit 3A). Anabelle entend des craquements dans la charpente. Lumière de la lucarne qui faiblit. Température qui chute. Bruit qui cesse puis reprend. Aucune source visible.

## 3B — Peur sociale

### Structure
La peur naît d'un **échange humain minimal** :
- **Dialogues laconiques** ou absents
- **Regards fuyants**, silences lourds
- **Sous-entendus** créant suspicion/malaise
- **Gestes figés** ou ambigus

### Contraintes
- Aucune confrontation explicite
- Tension psychologique uniquement
- Réalisme des interactions sociales

### Exemple d'utilisation
**Prompt :**
> Scène de retour du père (gabarit 3B). Il entre, voit les bottes d'Eva manquantes. Regarde Anabelle. Silence. Elle détourne les yeux. Il pose une question simple. Elle répond à côté. Fin de scène sans résolution.

## 5A — Questions en creux

### Structure
Générer **3 questions philosophiques/morales** sans discours direct :
- Les questions émergent d'**objets, gestes, regards**
- Aucun personnage ne les formule verbalement
- Le spectateur les ressent par l'accumulation de détails

### Contraintes
- Pas de voix-off explicative
- Pas de dialogue métaphysique
- Questions implicites uniquement

### Exemple d'utilisation
**Prompt :**
> Scène où Anabelle regarde les bottes d'Eva (gabarit 5A). Questions en creux : Aurait-elle pu empêcher la sortie? Le voisin savait-il? La tempête était-elle prévisible? Montrer via : main qui touche les bottes, regard vers la fenêtre, calendrier météo visible.

## 6A — Révélation partielle

### Structure
Dévoiler un **d��tail qui change le sens** sans confirmation :
- **Indice visuel** : objet, trace, document
- **Pas d'explication immédiate**
- **Deux interprétations possibles** laissées ouvertes

### Contraintes
- Ambiguïté maintenue
- Pas de résolution dans la scène
- Indice suffisamment clair pour être remarqué

### Exemple d'utilisation
**Prompt :**
> Révélation partielle (gabarit 6A) : le corps d'Eva apparaît sous la neige. On voit des cheveux, une main. Impossible de déterminer si c'est un accident ou un acte. Coupe avant que le père n'arrive.

## 6B — Révélation en trois temps

### Structure
Dévoilement progressif en **3 étapes** :
1. **Indice visible** : le spectateur voit un détail
2. **Indice compris** : un personnage relie les éléments
3. **Indice insupportable** : la pleine signification apparaît → coupe

### Contraintes
- Chaque temps doit être distinct
- Montée émotionnelle progressive
- Coupe avant catharsis complète

### Exemple d'utilisation
**Prompt :**
> Compréhension finale (gabarit 6B). Temps 1 : le père voit un ticket de déneigement daté du jour de la disparition. Temps 2 : il regarde le ponceau, comprend qu'Eva était sous la route. Temps 3 : il trouve une botte coincée dans le métal. Coupe.

## 7B — Découverte d'un objet

### Structure
Scène centrée sur **1 objet** révélant une contradiction émotionnelle :
- **3 détails physiques** : texture, couleur, usure
- **Réaction du personnage** : gestes, respiration, regard
- **Tension non résolue** : l'objet pose une question

### Contraintes
- Objet concret, tangible
- Pas de flashback automatique
- Détails sensoriels précis

### Exemple d'utilisation
**Prompt :**
> Découverte du journal (gabarit 7B). Objet : cahier bleu rigide. Détails : coins écornés, pages jaunies, écriture d'enfant. Anabelle le serre, tremble, lit une page. Bruit en bas. Fin sans révélation du contenu.

## 8 — Montage TV nouvelles

### Structure
Séquence de style **reportage** (2 minutes) :
- **Voix-off neutre** d'ancre TV
- **Témoignages courts** de villageois
- **Images d'archive** : météo, recherches
- **Détail discordant** en conclusion

### Contraintes
- Ton journalistique réaliste
- Pas de musique dramatique
- Information factuelle uniquement

### Exemple d'utilisation
**Prompt :**
> Montage TV (gabarit 8) sur l'enquête. Ancre : "Eva, 12 ans, disparue depuis 3 jours". Témoins : voisin, enseignante, policier. Images : tempête, forêt, maison. Détail final : une botte retrouvée à 2 km, direction opposée.

## 9A — EVA Validation strict

### Structure
Gabarit de **validation de scène** contre chronologie et continuité :
- **Vérification temporelle** : cohérence avec CHRONOLOGIE.md
- **Détection d'ambiguïtés** : dates, lieux, objets
- **Contrôle stylistique** : réalisme, hors-champ, suggestion
- **Sortie structurée** : OK / Ambiguïtés / Recommandations

### Contraintes
- Aucune réécriture automatique
- Signalement explicite des écarts
- Format décisionnel clair

### Exemple d'utilisation
**Prompt :**
> Valider la scène "Découverte du journal" avec le gabarit 9A contre CHRONOLOGIE.md et CONTINUITÉ.md. Vérifier : datation du journal, hors-champ respecté, réalisme du bruit en bas.

---

## Utilisation des gabarits

1. Choisir le gabarit adapté à la scène
2. Formuler un prompt incluant : gabarit + lieu/temps + contraintes spécifiques
3. Générer la scène avec EVA Scenario
4. Valider avec gabarit 9A (EVA Validation)
5. Ajouter les notes de continuité à CONTINUITÉ.md
