### Structure Globale
- **Format:** 240 segments de 30 secondes

## TRANSITIONS & TECHNIQUES VISUELLES

### Palette de Transitions

#### Transitions Narratives
1. **Cut Standard** - Segments 40% 
2. **Dissolve/Fade** - Segments 25%
3. **Match Cut** - Segments 15%
4. **Wipe Directionnel** - Segments 10%
5. **Glitch/Digital** - Segments 5%
6. **Morphing** - Segments 5%

#### Transitions Temporelles
- **Flashback Entry:** Glitch effect + desaturation
- **Flashback Exit:** Chromatic aberration correction
- **Time-lapse:** Speed ramping progressif
- **Slow Motion:** 120fps to 24fps pour moments clés

#### Transitions Thématiques
- **Humain → Cyborg:** Morphing organique/mécanique
- **Réel → Virtuel:** Digital pixelization
- **Passé → Présent:** Color grading shift
- **Public → Privé:** Iris in/out

### Code Couleur Narratif

| Palette | Contexte | Segments Typiques |
|---------|----------|------------------|
| **Cyan/Magenta** | Monde cyberpunk urbain | 11-25, 81-100 |
| **Warm Orange** | Moments humains/intimes | 26-40, 91-100, 235-240 |
| **Cold Blue** | Technologie/laboratoire | 1-10, 149-155 |
| **Red Alert** | Danger/conflit | 106-125, 201-225 |
| **Desaturated** | Flashbacks/mémoires | 5, 136-142 |
| **Mixed Neon** | Chaos urbain | 13-15, 66-70 |

### Techniques de Cadrage

#### Acte I (Segments 1-60)
- **Dominance:** Plans moyens et rapprochés
- **Mouvement:** Caméra fluide, découverte
- **Ratio:** 2.39:1 cinémascope

#### Acte II (Segments 61-180)
- **Dominance:** Variété - larges pour action, serrés pour drame
- **Mouvement:** Dynamique, handheld pour tension
- **Ratio:** 2.39:1 maintenu

#### Acte III (Segments 181-240)
- **Dominance:** Plans larges épiques + gros plans émotionnels
- **Mouvement:** Stabilisé pour action, fixe pour résolution
- **Ratio:** 2.39:1, possible IMAX pour segments 207-213

## NOTES DE PRODUCTION

### Organisation par Phases

#### PRÉ-PRODUCTION (Mois 1-3)

**Mois 1: Développement**
- Finalisation script segment par segment
- Storyboard complet 240 segments
- Casting principal et secondaire
- Location scouting
- Budget détaillé par séquence

**Mois 2: Préparation Technique**
- Design production (décors, costumes, props)
- Planification VFX - segments identifiés
- Répétitions acteurs principaux
- Tests caméra et éclairage
- Chorégraphie stunts séquences 41-60, 207-213

**Mois 3: Finalisation**
- Construction décors principaux
- Fabrication costumes et prothèses
- Casting figurants
- Réservation équipements
- Planning tournage optimisé

#### PRODUCTION (Mois 4-7)

**Stratégie de Tournage:**
- **Par blocs de séquences** (non chronologique)
- **Par location** (optimisation logistique)
- **Ratio visé:** 12-15 segments/jour
- **Durée estimée:** 16-20 jours de tournage effectif

**Ordre de Tournage Suggéré:**

1. **Bloc 1 (5 jours):** Segments intérieurs laboratoire/médical
   - Segments 1-10, 26-32, 149-155
   - Location: Studio décor labo

2. **Bloc 2 (4 jours):** Séquences urbaines extérieures
   - Segments 11-25, 66-70, 81-90
   - Location: City streets nuit

3. **Bloc 3 (3 jours):** Scènes intimistes
   - Segments 91-100, 131-135, 235-240
   - Location: Appartement set

4. **Bloc 4 (2 jours):** Séquences enquête/conspiration
   - Segments 111-125, 156-168
   - Locations: Multiples intérieurs

5. **Bloc 5 (4 jours):** Séquences action majeures
   - Segments 41-60, 181-200, 207-225
   - Location: Studio + extérieurs contrôlés

6. **Bloc 6 (2 jours):** Inserts, VFX plates, B-roll
   - Compléments divers segments
   - Multiples locations

#### POST-PRODUCTION (Mois 8-12)

**Mois 8-9: Montage**
- Assembly cut
- Rough cut séquence par séquence
- Fine cut avec timing précis 30 sec/segment
- Locked cut pour VFX

**Mois 10-11: VFX & Audio**

### Considérations Techniques Spéciales

#### Caméra & Format
- **Caméra recommandée:** RED Komodo 6K ou ARRI Alexa Mini
- **Format:** 6K pour flexibilité VFX
- **Codec:** RAW ou ProRes 4444
- **Frame rates:** 24fps standard, 60-120fps action
- **Lentilles:** Anamorphiques pour look cinéma

#### Éclairage Signature
- **LUTs personnalisées** par acte
- **Néons pratiques** pour cyberpunk
- **Volumétrique** pour laboratoire
- **Naturel augmenté** pour intimité

#### VFX Pipeline
- **Software:** After Effects, Nuke, Houdini
- **3D:** Blender/Maya pour assets cyborg
- **Compositing:** 32-bit pipeline
- **Render farm:** Cloud ou local selon budget

#### Audio Post
- **DAW:** Pro Tools
- **Plugins:** iZotope, FabFilter, Soundtoys
- **Libraries:** SFX cyborg custom + commercial
- **Mixing:** Dolby Atmos si budget permet

### Défis de Production Identifiés

1. **Cohérence visuelle** sur 240 segments
   - Solution: Bible visuelle stricte, LUTs préparées

2. **Performance cyborg crédible**
   - Solution: Ateliers acteur, tests maquillage extensifs

3. **Budget VFX limité**
   - Solution: Priorisation segments clés, pratiques quand possible

4. **Tournage efficace**
   - Solution: Storyboard précis, prévis 3D séquences complexes

5. **Continuité narrative**
   - Solution: Script supervisor rigoureux, base de données photos
