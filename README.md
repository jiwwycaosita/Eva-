# EVA - Dossier de Production Cinématographique

**Genre** : Thriller psychologique • Horreur réaliste  
**Durée** : 60 minutes  
**Format** : 2.35:1 (Cinémascope)  
**Langue** : Français québécois

## Synopsis

EVA est un projet cinématographique ambitieux qui s'inscrit dans la tradition des thrillers psychologiques et de l'horreur réaliste, avec une dimension dramatique et surnaturelle subtile. Le film explore la descente aux enfers d'une mère, Anabelle, dans un village isolé du Québec, après la disparition inexpliquée de sa fille Eva.

L'histoire se déroule dans un village québécois isolé, où une mère sombre dans la folie après la disparition de sa fille. Le film mêle réalisme cru, ambiance sonore immersive, et un surnaturel ambigu centré sur un chat noir mystérieux.

## Éléments Clés

- **Durée** : Film de 60 minutes
- **Structure** : 5 actes, 19 scènes détaillées
- **Thèmes** : Maternité, culpabilité, deuil, indifférence sociale, folie
- **Atmosphère** : Hiver québécois, silence oppressant, tension psychologique croissante
- **Personnages principaux** : Anabelle (mère), Eva (fille), Shawn (père), le Chat Noir (entité mystérieuse)

## Production

Le film est conçu pour une production indépendante avec un budget modeste, privilégiant l'atmosphère et la performance des acteurs plutôt que les effets spéciaux. Le tournage hivernal au Québec est essentiel pour l'authenticité visuelle et émotionnelle du projet.
