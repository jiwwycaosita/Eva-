# Eva-
EVA est un projet cinématographique ambitieux qui s’inscrit dans la tradition des thrillers psychologiques et de l’horreur réaliste, avec une dimension dramatique et surnaturelle subtile. Le film explore la descente aux enfers d’une mère, Anabelle, dans un village isolé du Québec, après la disparition inexpliquée de sa fille Eva.
